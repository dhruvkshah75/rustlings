## About this Repo

This project is based on the official [rust-lang/rustlings](https://github.com/rust-lang/rustlings) curriculum. The goal is to solve various programming challenges that cover the core concepts of Rust, from basic variables to advanced topics like threads and macros.

## Structure

* `exercises/`: The core challenges divided by topic (e.g., variables, functions, ownership).
* `solutions/`: My completed versions of the exercises.
* `Cargo.toml`: Project configuration and dependencies.
